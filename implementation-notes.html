Implementation Notes

The portfolio is a single-page site built with Bootstrap 5 components and features with a custom black, white and blue theme. This page documents:

  • Bootstrap features and components used throughout the site
  • Custom JavaScript features developed for enhanced functionality
  • Hosting details

1. Bootstrap Components and Features Used

This section lists Bootstrap features that provide layout, components and behaviour without custom implementation.

1.1 Interactive Components

Modal

  • Email contact modal in the Contact section with form inputs
  • Uses .modal, .modal-dialog, .modal-content, .modal-header, .modal-body, and .modal-footer
  • Triggered with data-bs-toggle="modal" and data-bs-target="#emailModal"
  • Includes .btn-close for accessible close functionality
  • Bootstrap handles opening, closing, backdrop, and focus management
  • Docs: Modal

Carousel

  • Three carousels in the Hobbies section (Music, Motorcycles, Hiking)
  • Uses .carousel, .carousel-inner, .carousel-item, .carousel-indicators, and .carousel-control-prev/next
  • Autoplaying functionality with data-bs-ride="carousel"
  • Navigation controls with previous/next buttons and slide indicators
  • Docs: Carousel

1.2 Navigation and Page Behaviour

Navbar

  • The top navigation uses .navbar, .navbar-expand-lg, .navbar-dark, .bg-dark, and .fixed-top
  • Includes .navbar-brand and .navbar-nav with section links
  • Collapses into a hamburger menu on smaller screens with .navbar-toggler
  • Used at the top of index.html
  • Docs: Navbar

Scrollspy

  • Enabled on <body> with data-bs-spy="scroll", data-bs-target="#mainNav", and data-bs-offset="80"
  • Bootstrap automatically updates the active navbar link based on scroll position
  • Docs: Scrollspy

Collapse

  • The mobile menu uses Collapse via data-bs-toggle="collapse" and data-bs-target="#navbarContent"
  • Bootstrap handles show and hide behaviour automatically
  • Docs: Collapse

1.3 Forms, Buttons and Icons

Buttons

  • Variety of button styles: .btn-primary, .btn-secondary, .btn-success, .btn-outline-light
  • Size variants: .btn-lg, .btn-sm
  • Full-width buttons with .w-100 in Contact section
  • Custom sharp corners (border-radius: 0) applied via CSS
  • Docs: Buttons

Forms

  • The email form inside the Contact modal uses Bootstrap form markup
  • Includes .form-label, .form-control, and custom .bg-black styling
  • Input types: text, email, and textarea
  • HTML5 validation with required attributes
  • Docs: Forms

Bootstrap Icons

  • Icons throughout the site from Bootstrap Icons library (CDN)
  • Examples: .bi-envelope-fill, .bi-github, .bi-linkedin, .bi-trophy-fill, .bi-music-note-beamed, .bi-robot
  • Used for visual enhancement in navigation, sections, achievements, skills, and social links
  • Docs: Bootstrap Icons

1.4 Content and Data Display

Card

  • Used extensively throughout the different sections
  • Components include .card, .card-body, .card-header with custom .bg-dark and .border-secondary styling
  • Height utilities like .h-100 ensure consistent card heights in project grid
  • Docs: Card

Badge

  • Bootstrap badges with .badge, .bg-secondary, .bg-primary, and .bg-info
  • Used to label skills (technical and soft skills), project technologies, education focus areas, and achievements
  • Custom border styling applied for sharp corners
  • Docs: Badges

Alerts

  • Alert component used in email modal with .alert, .alert-info
  • Informs users that clicking send will open their default email app
  • Docs: Alerts

1.5 Layout and Structure

Grid System and Containers

  • .container, .row, .col-md-6, and responsive classes such as .col-md-6 and .row.g-4 control the layout in the hero, about, education, projects, experience, skills, hobbies and contact sections.
  • Custom max-width of 65rem applied to containers for consistent page width
  • Docs: Containers, Grid

Utility Classes

  • Spacing classes such as .mb-4, .pt-4, .py-5, and .gap-2
  • Flex utilities such as .d-flex, .justify-content-between, .align-items-center, .flex-column, .flex-md-row
  • Text utilities: .text-center, .text-muted, .text-md-end, .text-white
  • Display utilities: .d-block, .d-inline-flex, .d-grid
  • Docs: Flex utilities, Spacing, Text

Responsive Images

  • Classes such as .w-100, .h-auto, and .d-block are used in the hero portrait and hobbies carousels to keep images responsive
  • Docs: Images

1.6 Additional Bootstrap Features

Border Utilities

  • Border classes like .border-secondary, .border-start, .border-bottom, .border-top
  • Used extensively for the custom bordered design aesthetic
  • Docs: Borders

Shadow Utilities

  • .shadow-sm applied to the fixed navbar
  • Docs: Shadows

List Groups

  • List styling with .list-unstyled used in Education and Experience sections
  • Custom bullet points using Bootstrap Icons instead of default list markers
  • Docs: List group

2. JavaScript Features

Custom JavaScript functionality implemented to enhance user interactions and experience.

Skills Toggle with Fade Animation

  • Custom toggle button switches between Technical Skills and Soft Skills sections
  • Smooth fade-in/fade-out animations using CSS
  • Animation lock prevents stuttering when clicking rapidly

Email Form Handler

  • Custom sendEmail() function collects form data
  • Validates all required fields before submission
  • Generates mailto: link with pre-filled subject and body
  • Opens user's default email client with formatted message
  • Automatically closes modal and resets form after send

Back to Top Functionality

  • Smooth scroll behavior using window.scrollTo() with behavior: 'smooth'
  • Event listener attached to footer button
  • Works across all pages (index.html and implementation-notes.html)
